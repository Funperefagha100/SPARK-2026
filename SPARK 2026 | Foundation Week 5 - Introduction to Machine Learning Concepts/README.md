## Overview

Welcome to Week 5 of SPARK Academy 2026! This week introduces core machine learning concepts — from supervised and unsupervised learning to practical implementation, and explores the key neuroimaging tools used in clinical and research workflows, including 3D Slicer, FreeSurfer, MONAI Label, and ITK.

## Session 2: Neuroimaging Tools and Applications

A live session providing an overview of the key tools used in neuroimaging workflows, from 3D visualisation to AI-assisted annotation and image analysis.

**Topics Covered:**
- Overview of neuroimaging workflows
- 3D Slicer for medical image visualisation
- FreeSurfer for morphometry
- MONAI Label for AI-assisted annotation and labeling of medical images
- ITK for image analysis
